Answer to Question 2 of Code Foo Challenge

Posted by Eric Sare on April 29, 2012 • Comments (64) • Full article

A liquid layout allows contents to expand or contract dynamically, texts on the page can reflow to accomodate the change in screen size as well.

This sample page demonstrates how a liquid design responds to the changes in screen size. The page will switch to a mobile version when the window width goes below 600 pixels. Please have fun resizing the window and see how the page changes!
